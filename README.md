# Binary to Decimal Converter

This project is a simple application that converts binary numbers to decimal format. It is built using C# and .NET.

## Features
- Convert binary numbers to decimal.
- User-friendly interface.

## Getting Started

### Prerequisites
- [.NET SDK](https://dotnet.microsoft.com/download) installed on your machine.

### Installation
1. Clone the repository:
   
   ~~~
   git clone https://github.com/yourusername/bin2dec-app.git
   ~~~

2. Navigate to the project directory:
   
   ~~~
   cd bin2dec-app
   ~~~

3. Restore the dependencies:
   
   ~~~
   dotnet restore
   ~~~

4. Run the application:
   
   ~~~
   dotnet run
   ~~~

## Usage
- Enter a binary number in the input field.
- Click the "Convert" button to see the decimal equivalent.
